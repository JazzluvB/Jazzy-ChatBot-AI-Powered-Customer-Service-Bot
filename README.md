# Jazzy-ChatBot-AI-Powered-Customer-Service-Bot
Jazzy ChatBot es un asistente virtual inteligente diseñado para la agencia Jozzy Creative Studio, especializada en edición de video y fotografía. El bot guía a los clientes potenciales a través de los servicios disponibles, recomienda paquetes según sus necesidades y los conecta con un asesor humano vía Discord cuando es necesario.
